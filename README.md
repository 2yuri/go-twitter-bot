#Como criar um bot do Twitter com go

```bash
github.com/dghubble/go-twitter
github.com/dghubble/oauth1
```

### Configuração:

1. Exportar variáveis de ambiente:

   ```bash
   CONSUMER_KEY
   CONSUMER_SECRET_KEY
   TOKEN_KEY
   TOKEN_SECRET_KEY
   ```

2. Criar o nosso client

## Funcionalidades:

- [x] Realizar tweet
- [x] Listar tweets por query
- [x] Retweet de algum tweet
