### Configuração:

1. Exportar variáveis de ambiente:

   ```bash
   CONSUMER_KEY
   CONSUMER_SECRET_KEY
   TOKEN_KEY
   TOKEN_SECRET_KEY
   ```

2. Criar o nosso client
